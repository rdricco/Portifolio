Sobre esse site

Site desenvolvido por Renato Ricco usando Jekyll, uma poderosa ferramenta de construção de sites estáticos. Assim como o Jekyll, o código fonte desse site é distribuido gratuitamente utilizando a licença MIT e pode ser personalizado de acordo com as suas necessidades tanto para uso pessoal quanto para uso comercial, desde que essa licença seja mantida. Crie seu fork no github ou contribua para melhorias.
